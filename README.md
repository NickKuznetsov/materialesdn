# Materiales del Norte Site

## Overview
The "Materiales del Norte" project is a website designed to showcase various materials and products offered by the company. This site aims to provide users with an intuitive interface to explore the offerings and learn more about the materials available.

## Project Structure
```
materiales-del-norte-site
├── index.html          # Main HTML document for the site
├── assets
│   ├── css
│   │   └── styles.css  # CSS styles for the website
│   └── img             # Folder for images (currently empty)
└── README.md           # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the site.
3. Modify the `styles.css` file in the `assets/css` folder to customize the appearance of the site.
4. Add images to the `assets/img` folder as needed.
